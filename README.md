# classhunter
```
Usage: classhunter [-hV] [-d=<searchDirectoryOption>] (-m | -j)
  -d, --directory=<searchDirectoryOption>
                  directory to search
  -h, --help      Show this help message and exit.
  -j, --jars      Search by jar
  -m, --modules   Search by module
  -V, --version   Print version information and exit.
```
